# Arch-Installation-Notes
This is a step by step guide that I use to install Arch Linux

## Initialization
check uefi boot mode
``` ls /sys/firmware/efi/efivars ```
check internet
``` ping google.com ```
update clock
``` timedatectl set-ntp true ```
## Partitioning
check the hard disks
``` fdisk -l ```
use cfdisk to partition
``` cfdisk /dev/sda ```
